# QingBoard

QingBoard（极简跨端无缝剪切板）是一款专为个人打造的轻量级、多平台通用的临时剪贴板服务工具。它提供了一个高颜值的网页操作界面，并支持像原生桌面 App 一样的物理键盘快捷键，让你能够在不同时间、不同设备、不同地点的电脑之间光速传递文字、图片及文件。

---

## ✨ 核心特性

- **跨端无缝接力：** 完全基于网页运作（Web-based），只要在家里部署好服务端，你便可以在任何地方的主机/手机浏览器里随时粘贴或提取数据。全客户端实时轮询同步，数据秒级刷新。
- **动态 8 宫格：** 界面默认包含 8 个并行的剪切槽位。响应式的现代 UI 会自动适配您的屏幕宽度。
- **高级质感 (Glassmorphism)：** 全局采用高质感的毛玻璃效果以及沉浸式暗色模式设计界面，提供优秀的视觉反馈及护眼使用体验。
- **全键盘支持：** 您可以通过物理键盘进行高效率操作：支持 `Ctrl+C/V/X`（Mac 上为 `Cmd+C/V/X`），支持按 `Delete` / `Backspace` 快速清空槽位，同时支持弹窗时按 `Enter` 确认、按 `Esc` 退出。
- **文件与图片的高级处理：**
    - 支持电脑桌面的“图片/文档”直接拖拽（Drag & Drop）直达槽位。
    - **原生剪贴板接驳**：网页提取图片时采用了特殊的预转码技术（图片转为最高画质原生 `image/png` 格式），你在 QingBoard 里点一下图片，再去 Word 里按 `Ctrl+V` ，可以直接粘贴出图片本体，不再是单薄的文本链接。

## 🚀 如何启动

程序已为您配置了极其简化的一键启动脚本。在项目所在目录下，打开终端并执行：

```bash
./run.sh
```

*(该脚本将自动使用后台的 Python 虚拟环境，并默认以 `16968` 端口启动服务。)*

启动成功后，在浏览器访问下面的网址即可进入主界面：
`http://127.0.0.1:16968` （您可以在其他设备上将 IP 替换为您这台电脑的局域网/公网 IP）。

## 📖 使用指南

为了追求极致简单的体验，每个剪切槽位只有 3 种交互按钮：
1. **复制 (`Copy`)**：将槽位里的文本或图片格式，瞬间提取到你当前所在电脑的系统剪切板中。*(也可以直接用鼠标点击卡片的任意空白发光处来极速盲操复制)*
2. **删除 (`Delete`)**：清空您不需要的临时文件，以释放格位。（出于安全设计，清空前会要求二次确认）
3. **上传 (`Upload`)**：当您需要将本地硬盘深处的文档上传到剪切板时，可以通过此按键开启文件管理器选择。


> **数据存储机制声明：** 
> *为了追求性能与轻量，您的文字记录仅存放于运行时的内存之中，重启服务后文字便会自行销毁。图片及文件则会安全地缓存于后端的 `uploads` 目录下。作为私人的跨端周转容器，请及时清理不再需要的内容。*
